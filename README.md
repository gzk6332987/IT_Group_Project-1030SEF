## Objective of our project

The aim of this Internet Application project is to develop an accessible, and user‑friendly webpage that combining various functions, e.g., academic results, attendance, and homework submissions. Login system will be included. a modern graphical user interface following WCAG standards will be provided to support disabled users. Comments will be added near the lines to ensure good readability and future maintenance.
